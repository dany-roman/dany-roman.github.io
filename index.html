Analysis of City Weather by Latitude


This project focused on analyzing the weather conditions across different Latitudes. Using data collected from the OpenWeatherMap API on May 24th 2021 for a set of over 500 cities worldwide, I used Matplotlib to compare Latitude by temperature (F), humidity (%), wind speed (mph), and cloudiness (%).

This site briefly describes the data, visualizations, and findings of this project. Click below to learn more about how latitude affects different weather conditions.
